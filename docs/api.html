API

Hooks/Methods

Renderers

Renderers have a number of hooks/methods available to structure them and call pieces of Javascript at key moments on navigation. Read more of the documentation about renderers.

Hooks/Methods Description
onEnter() This method is run when the data-router-view is added to the DOM Tree.
onLeave() This method is run when transition to hide the data-router-view is called.
onEnterCompleted() This method is run when the transition to display the data-router-view is done.
onLeaveCompleted() This method is run when the data-router-view is removed from the DOM Tree.

Transitions

Transitions have a number of hooks/methods available to manage which animations should be run to display a page and which one should be run to hide it. Read more of the documentation about transitions.

Methods Parameters Description
in(view, done)

view | <Node>

done | <Callback> 		This method is called to display the data-router-view.
out(view, done)

view | <Node>

done | <Callback> 		This method is called to hide the data-router-view.

Core

In addition to the methods inherited from tiny-emitter, we have some buil-in ones available we might need:

Methods Parameters Description
Core.attach() This method listen to all links of the document that have no target or data-router-disabled attribute and calls Highway on click.
Core.detach() This method clears the events attached to all the links of the document.
Core.redirect(href) This method navigates to a given URL using Highway.

Events

Highway extends tiny-emitter so Highway.Core exposes all the methods available in tiny-emitter to use the events below.

Events Parameters Description
NAVIGATE_IN This event is emitted everytime the transition to show the data-router-view is called. It returns those arguments:

  • to: The data-router-view we show.
  • location: The URL datas of the current view.
NAVIGATE_OUT This event is emitted everytime the transition to hide the data-router-view is called. It returns those arguments:

  • from: The data-router-view we hide.
  • location: The URL datas of the current view.
NAVIGATE_END This event is emitted everytime a transition ends. It returns those arguments:

  • to: The data-router-view we show.
  • from: The data-router-view we hide.
  • location: The URL datas of the current view.
